# MHHPM
project movie

## server
  - yarn install
  - yarn start / dev
  - port của server: localhost:1000
  - Nếu muốn chỉnh port chạy ở server thì vô server/src/config/index.js chỉnh sửa value port ở đây

## customers
  - yarn install
  - yarn start
  - port auto của reactjs
  - nếu server đổi khác port 1000 thì vô customers/src/api/api.js đổi baseURL thành port của server đang chạy
